## スキルチェック問題 一日決められた項目をやる 平均時間内に解くことを目標にする

### 問題を解くにあたって注意すること

- まず問題をよく読んで意味を理解する

- 次にどういう処理をするコードを使うか考える

- 入力が終わったら実行して確認

- エラーがあればエラーの箇所と何が原因かをわかるようにする

- 訂正もしくはコードを書き直す 確認してできるまで繰り返す
